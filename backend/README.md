# Tastivo Backend

Basic Express API scaffold for Tastivo.

## Setup

1. Copy `.env.example` to `.env` and adjust values.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm run dev
   ```
   or for production
   ```bash
   npm start
   ```

## Endpoints

- `GET /health` – uptime/status check.
- `GET /api/hello` – placeholder endpoint to confirm the API is running.
